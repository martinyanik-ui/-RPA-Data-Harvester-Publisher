# RPA: Automatización de Consultas Públicas y Seguimiento de Expedientes

## 📋 Descripción
Este proyecto resuelve la carga operativa de consultar manualmente el estado de expedientes y licitaciones en portales de administración pública. Mediante **Power Automate Desktop**, el bot realiza búsquedas automáticas, extrae estados, fechas y montos, y genera un reporte estructurado para su posterior análisis.

## ⚙️ Funcionalidades
- **Navegación Autónoma:** Login (si aplica) y búsqueda por parámetros dinámicos.
- **Web Scraping Avanzado:** Captura de tablas dinámicas con manejo de paginación.
- **Tratamiento de Datos:** Limpieza de strings y conversión de formatos monetarios/fechas.
- **Integración:** Generación de archivos CSV/Excel compatibles con Power BI.

## 🛠️ Tecnologías
- **Herramienta:** Power Automate Desktop
- **Lenguaje:** Selectores CSS y UI Automation
- **Destino:** Excel / CSV

## 📊 Impacto en la Gestión
1. **Transparencia:** Seguimiento diario sin depender de reportes manuales.
2. **Eficiencia:** Reducción del tiempo de consulta en un 90%.
3. **Análisis de Datos:** Permite identificar cuellos de botella en los tiempos de tramitación administrativa.
